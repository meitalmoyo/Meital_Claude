## Common Impasse Types and Responses

| Impasse type | What it looks like | Possible response |
|-------------|-------------------|-------------------|
| Positional lock | Both parties repeat their demands | Reality test each separately in caucus |
| Emotional flooding | One or both parties too activated to engage | Break, caucus, slow down |
| Trust collapse | Neither party believes the other will honor an agreement | Focus on what verification looks like |
| Value conflict | Core values incompatible | Shift from agreement to coexistence — what can be managed? |
| Information gap | Parties working from different facts | Identify what facts are needed, how to get them |
