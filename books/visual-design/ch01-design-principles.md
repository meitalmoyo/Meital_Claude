# Visual Design — ch01: Design Principles

Core visual foundations that apply across all MOYO Productions work.

---

## The 7 Principles

**Contrast** — Make differences clear. Never use similar colors, sizes, or weights when you need to distinguish elements. Contrast creates hierarchy.

**Hierarchy** — Guide the eye. Most important element is biggest/boldest/highest. The viewer should never have to decide where to look first.

**Alignment** — Nothing is placed arbitrarily. Every element aligns to something. Invisible grid = professional result.

**Repetition** — Repeat visual elements (color, font, shape, line) to create consistency. Repetition = brand recognition.

**Proximity** — Related items go together. Unrelated items have space between them. Grouping communicates relationship.

**Balance** — Symmetrical (formal, stable) or asymmetrical (dynamic, modern). Both work — pick one and commit.

**White space** — Not empty space, active space. It gives the eye rest and makes everything else breathe.

---

## Color

**CMYK for print, RGB for screen** — never mix them in the same file without converting.

**Color psychology (Israeli market context):**
- Blue → trust, professionalism (finance, legal, mediation)
- Green → growth, health, nature
- Red → urgency, energy, danger (use sparingly)
- White → clean, minimal, modern
- Gold/Warm tones → premium, warmth, approachability

**Contrast ratios for accessibility** — text on background needs to pass readability at small sizes. Test it.

**Brand color discipline** — When a client has a brand color, use it. Don't "improve" it. Your job is consistency, not reinvention.

---

## Typography

**Maximum 2 typefaces per design** — one for headlines, one for body. Three is pushing it. Four is chaos.

**Hierarchy with type:** Size → Weight → Color → Case. Use one at a time before stacking them.

**Serif vs Sans-serif:**
- Serif: traditional, authoritative, print-friendly (books, formal docs)
- Sans-serif: modern, clean, screen-friendly (web, app, presentations)

**Line length** — 60–75 characters per line for body text. Longer = harder to read.

**Leading (line spacing)** — 1.4–1.6× the font size for body. Headlines can be tighter.

---

## Composition

**Rule of thirds** — divide the canvas into 3×3 grid. Place key elements at intersections.

**Visual weight** — dark colors, large shapes, and complex elements feel heavy. Balance them with lighter elements.

**Entry point** — where does the eye enter the composition? Usually top-left (Hebrew: right-to-left, so consider the flip).

**Hebrew design note** — RTL text requires mirrored layouts. Don't just flip an LTR design — rebuild it. Navigation flows right to left. Reading direction affects where the eye enters.
