# Visual Design — ch07: Branding

Logo, identity systems, and brand guidelines.

---

## What Branding Is (and Isn't)

**Branding is a system**, not a logo. A logo is one element. A brand is everything the client's audience sees and feels — colors, fonts, tone of voice, imagery style, shapes, spacing.

**A logo can't save a bad brand.** And a great brand can work with a simple logo.

---

## Logo Design

**Start with concepts, not software.** Sketch 10–20 thumbnail ideas on paper before opening Illustrator. The best idea is rarely the first idea.

**Logo must work:**
- In full color
- In black and white (one color)
- Reversed (white on dark background)
- At very small sizes (favicon, stamp)
- At very large sizes (billboard, banner)

**Types:**
- Wordmark — company name in custom typography (clean, timeless)
- Lettermark — initials only (works when name is long)
- Icon/Symbol — standalone graphic (takes time to build recognition)
- Combination mark — icon + wordmark (most versatile for new brands)

**File deliverables for logo:**
- `.ai` — master vector source
- `.eps` — universal vector format
- `.svg` — web use
- `.png` (transparent background) — general use, multiple sizes
- `.jpg` — when transparency isn't needed

## Brand Color System

**Primary palette:** 1–2 main brand colors
**Secondary palette:** 2–3 supporting colors
**Neutral palette:** 1–2 neutrals (white, off-white, dark, grey)

**Document every color in:**
- HEX (web)
- RGB (digital)
- CMYK (print)
- Pantone (if applicable)

**Don't just document the colors — document the usage rules.** Which color is for headlines? CTAs? Backgrounds? Without rules, the brand drifts.

---

## Typography in Branding

**Primary typeface** — for headlines, large text, brand voice
**Secondary typeface** — for body text, smaller elements

Rules to define:
- Font sizes for each usage (H1, H2, body, caption)
- Line height
- Letter spacing (tracking)
- Weight variations (bold for X, regular for Y)

**Hebrew branding note:** If the brand is bilingual (Hebrew + English), you need a font pair that works in both languages. Not all Latin fonts have Hebrew character sets. Common safe pairings:
- Heebo (Hebrew) + Poppins (English)
- Rubik (Hebrew) + Rubik (English) — Rubik supports both
- Assistant (Hebrew) + Nunito (English)

---

## Brand Guidelines Document

Minimum for a small client:
1. Logo usage (do's and don'ts, clear space rules)
2. Color palette with values
3. Typography rules
4. Example applications (business card, letterhead, social post)

**Format:** PDF, clean and printable. Not a 60-page book — a practical reference they'll actually use.
